# KickBlastStableLight

Simple WPF .NET 8 app using code-behind and SQLite (Microsoft.Data.Sqlite).

## Setup
1. Create a WPF App (.NET 8) in Visual Studio named `KickBlastStableLight`.
2. Install NuGet package `Microsoft.Data.Sqlite` version `8.0.0`.
3. Copy these files into the project.
4. Run.
